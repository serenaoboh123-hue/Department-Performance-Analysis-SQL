# Department Performance Analysis Using SQL

## Project Overview
This project analyzes employee, department, and performance data using SQL to generate workforce insights. The analysis combines multiple tables to answer business questions related to employee performance, departmental structure, hiring trends, and salaries.

## Objectives
- Analyze employee performance across departments.
- Explore workforce and hiring trends.
- Join multiple relational tables to generate business insights.
- Demonstrate practical SQL skills for data analysis.

## Dataset
The project uses three related datasets:
- Department
- Staff
- Performance

## SQL Skills Demonstrated
- INNER JOIN
- USING Clause
- WHERE Filtering
- Aggregate Functions
- YEAR() Function
- Table Relationships
- Multi-table Analysis

## Sample Business Questions
- Which employees belong to each department?
- How do employee performance scores vary across departments?
- What are the hiring years of employees?
- Which departments have the highest-performing employees?
- How are salaries distributed across different regions?
